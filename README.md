# ColdReach

ColdReach is a job outreach platform that helps you find and contact Y Combinator startups directly.

The app keeps an up-to-date list of YC companies, synced automatically every 6 hours, so you always have fresh data on which companies exist, which ones are hiring, and basic details like their batch, team size, and description.

You can search and filter through the list, and the goal is to eventually let you draft and send cold emails to these companies right from the dashboard, without needing a spreadsheet or manual research.

## Status

This project is currently under development. Features are being added and things may change or break.

## Tech stack

- Frontend: React, TypeScript, Redux Toolkit, Tailwind CSS
- Backend: Node.js, Express, TypeScript, MongoDB, Redis
